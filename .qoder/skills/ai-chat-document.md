# AI Chat & Document Recognition Skill

## 功能概述

这是一个综合性的 AI 技能，提供智能聊天和文档识别两大核心功能。

## 核心能力

### 1. AI 智能聊天

#### 功能特性
- **多轮对话**：保持上下文记忆，支持连续对话
- **意图识别**：准确理解用户需求
- **知识问答**：回答各类问题
- **代码辅助**：编写、解释、调试代码
- **多语言支持**：中文、英文等

#### 使用场景
```
用户：你好，能帮我解释一下这段代码吗？
AI：当然可以！请把代码发给我，我会详细解释。

用户：function hello() { console.log("Hello"); }
AI：这是一个 JavaScript 函数...
```

### 2. AI 文档识别

#### 功能特性
- **文本提取**：从文档中提取文本内容
- **格式分析**：识别文档结构和格式
- **内容摘要**：生成文档摘要
- **智能问答**：基于文档内容回答问题
- **关键信息提取**：提取重要数据和信息

#### 支持的文档类型
- 纯文本文件 (.txt)
- Markdown 文件 (.md)
- 代码文件 (.js, .ts, .py, .java 等)
- 配置文件 (.json, .yaml, .xml 等)
- 日志文件 (.log)

#### 使用场景
```
用户：请帮我分析这个文件的内容
AI：我来分析一下...

用户：这个文档主要讲了什么？
AI：根据文档内容，主要包含以下几个要点...
```

## 使用方法

### 基本聊天
直接在输入框中输入问题即可，AI 会自动识别意图并回复。

### 文档分析
1. 上传或粘贴文档内容
2. 提出关于文档的问题
3. AI 会基于文档内容进行分析和回答

### 代码辅助
1. 发送代码片段
2. 说明需求（解释、优化、debug 等）
3. AI 提供详细的技术支持

## 配置参数

```json
{
  "temperature": 0.7,
  "max_tokens": 2000,
  "context_window": 4096,
  "stream": true
}
```

## 最佳实践

### 提高回答质量
1. **问题明确**：尽量提供清晰、具体的问题
2. **上下文完整**：提供必要的背景信息
3. **分步提问**：复杂问题分步骤询问
4. **补充信息**：根据 AI 的回复补充细节

### 文档处理技巧
1. **分段处理**：大文档分段分析效果更好
2. **指定重点**：明确指出关注的部分
3. **提供目标**：说明分析的目的

## 注意事项

- 确保 Ollama 服务正常运行
- 选择适合任务的模型
- 长文档可能需要更长的处理时间
- 涉及敏感信息时注意隐私保护
